# 📓 J-Journal

A lightweight, local-first Journaling Application built with **Java**. Designed for privacy and simplicity, J-Journal allows you to document your daily thoughts, track your mood, and manage your entries through a clean, intuitive interface.

---

## 🚀 Features

*   **📝 Rich Text Entries:** Create, edit, and delete daily journal logs.
*   **📅 Calendar View:** Navigate through your history using a built-in calendar interface.
*   **🔐 Privacy Focused:** All data is stored locally on your machine—your thoughts stay yours.
*   **📂 Export Options:** Save your entries as `.txt` or `.pdf` for easy backup.
*   **🔍 Global Search:** Instantly find past entries using keywords or dates.
*   **🌓 Dark Mode:** Optimized for late-night writing sessions.
